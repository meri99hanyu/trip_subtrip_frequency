# Quick Start
## Download the datasets
- subtrip_roads.csv - https://drive.google.com/file/d/10-Co0L_kEILZYc_t-fkVeFYbx2ASF-pP/view?usp=sharing
- trip_roads.csv - https://drive.google.com/file/d/10-Co0L_kEILZYc_t-fkVeFYbx2ASF-pP/view?usp=sharing

## Run the script
```
python3 main.py
```

The output will be saves as "trip_roads_city_frequency.csv" for trips and "trip_roads_city_frequency.csv" for sub trips.
